# EasyEDA Pro iBOM 扩展 - 日志收集

## 请执行以下步骤获取日志：

### 方法 1: 在 EasyEDA 控制台直接获取（推荐）

1. 在 EasyEDA Pro 中按 `F12` 打开开发者工具
2. 切换到 **Console** 标签
3. 复制所有可见的日志内容
4. 粘贴到聊天中

或者在控制台中运行：
```javascript
// 获取 iBOM 日志
window.__IBOM_GET_LOGS__()

// 或者获取完整日志缓冲区
window.__IBOM_LOG_BUFFER__
```

### 方法 2: 启动桥接服务器后获取

1. 启动桥接服务器：
```bash
node C:\Users\AOC\.qwen\skills\easyeda-api\scripts\bridge-server.mjs
```

2. 在后台运行（新窗口）：
```bash
start /B node C:\Users\AOC\.qwen\skills\easyeda-api\scripts\bridge-server.mjs
```

3. 验证连接：
```bash
curl http://localhost:49620/health
```

4. 执行代码获取日志：
```bash
curl -X POST http://localhost:49620/execute ^
  -H "Content-Type: application/json" ^
  -d "{\"code\": \"return window.__IBOM_GET_LOGS__();\"}"
```

### 方法 3: 使用调试脚本

```bash
cd D:\easyeda-eext\eext-interative-bom\debug
start-debug.bat
```

## 需要的日志内容

请提供以下日志：

### 1. 扩展加载日志
```
[iBOM] Extension activated
[iBOM] ========================================
[iBOM] showInteractiveBom() called
```

### 2. PCB 文档检查日志
```
[iBOM] Is PCB document: true documentType: 3
[iBOM] PCB document active: true
```

### 3. PCB 数据获取日志
```
[iBOM] PCB source obtained: success
[iBOM] BOM items built: X
[iBOM] Project info: {...}
```

### 4. 错误日志（如果有）
```
[iBOM] showInteractiveBom() ERROR: ...
```

### 5. 完整控制台输出

包括所有 `[iBOM]` 前缀的日志和任何错误堆栈。

## 快速获取命令

在 EasyEDA 控制台中运行：
```javascript
// 获取并格式化日志
console.log('=== iBOM Logs Start ===');
console.log(window.__IBOM_GET_LOGS__());
console.log('=== iBOM Logs End ===');
```

然后复制输出结果。
